If you choose to update pool settings, you have the following things to adjust. Changes to the individual parameters are optional, so you don’t have to change them all.

Pool status:

     •  Open pool: open a pool for a previously closed validator
     •  Closed for new: close the pool to new delegators. Existing delegators are not affected.
     •  Close pool: close a pool for all delegators.

Metadata URL:

     •  The metadata URL can be changed, removed, or left the same.
